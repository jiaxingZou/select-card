# 免费部署静态网站指南

## 方法一：Netlify Drop（最简单，推荐）⭐

### 步骤：
1. 访问 https://app.netlify.com/drop
2. 直接将整个项目文件夹（包含 index.html, script.js, styles.css, boss-list.js）拖拽到网页上的拖拽区域
3. 等待上传完成，Netlify 会自动为你生成一个网址（格式类似：https://random-name-12345.netlify.app）
4. 完成！你现在可以通过这个网址访问你的网站了

**优点**：无需注册账号，无需安装任何工具，最快最方便

---

## 方法二：GitHub Pages（需要 GitHub 账号）

### 步骤：
1. 注册 GitHub 账号（如果还没有）：https://github.com/signup
2. 在 GitHub 上创建一个新仓库（Repository）
3. 将你的所有文件上传到仓库
4. 在仓库设置中启用 GitHub Pages，选择主分支（main 或 master）
5. 访问你的网站：https://你的用户名.github.io/仓库名

**优点**：永久免费，可以版本控制，专业

---

## 方法三：Vercel（也很简单）

### 步骤：
1. 访问 https://vercel.com
2. 使用 GitHub 账号登录（或注册新账号）
3. 点击 "Add New Project"
4. 直接拖拽你的项目文件夹
5. 自动部署完成，获得网址

**优点**：速度快，CDN 加速，免费

---

## 方法四：Cloudflare Pages

### 步骤：
1. 访问 https://pages.cloudflare.com
2. 登录 Cloudflare 账号（免费注册）
3. 创建新项目，上传文件
4. 获得网址

**优点**：全球 CDN，速度快

---

## 推荐使用方法一（Netlify Drop）

这是最简单的方法，只需要：
1. 打开浏览器
2. 访问 https://app.netlify.com/drop
3. 把包含所有文件的文件夹拖进去
4. 完成！

网址会立即生效，你可以分享给任何人使用。
